Client

Comment installer BeamMP ?

  • Nous avons un guide complet pour installer BeamMP sur Windows, vous pouvez le trouver ici.

BeamMP fonctionne-t'il sur une version crackée de BeamNG.drive ?

  • Non, BeamMP ne fonctionne pas sur les versions crackées ni sur les versions qui ne sont pas à jour du jeu. Si vous voulez jouer à BeamMP, vous pouvez acheter le jeu sur Steam.

BeamMP fonctionne-t'il sur Linux ?

  • Le client BeamMP ne fonctionne pas officiellement sur Linux, cependant il est possible de le faire fonctionner avec Wine.
  • Le serveur peut être build sur Linux. Cependant les builds ne sont pas fournies, vous devrez le faire vous même. BeamMP est officiellement supporté sur Debian 9 et 10, un tutoriel peut être trouvé ici.

Pourquoi le launcher est détecté comme un virus par Windows Defender ou mon antivirus ?

  • Afin d'empêcher Windows Defender de détecter le launcher comme un virus, l'achat d'une license est obligatoire, c'est dernière sont très coûteuses. Pour cette raison BeamMP pourrait être détecté comme un virus.
  • Certains antivirus pourraient détecter BeamMP comme un virus car ce dernier intérargit avec le réseau. Cela est normal, si vous avez un doute le code source complet peut-être retrouvé sur notre Github.

Les performances sont mauvaises, que puis-je faire ?

  • Nous faisons notre meilleur afin de conserver les meilleures performances possible. Cependant il n'y a pas grand chose qui puisse être fait, si vos performances sont toujours mauvaises avec des graphismes réduits, nous vous invitons à jouer sur un serveur avec moins de joueurs. Le jeu est principalement affecté par le processeur, si votre processeur possède peu de coeurs (4 ou moins) nous vous invitons à eviter les serveurs avec trop de joueurs.

J'ai découvert un bug / une faille / un exploit, que dois-je faire ?

  • Si le problème est causé par le code et que vous avez un compte Github, vous pouvez poster un ticket sur notre Github.
