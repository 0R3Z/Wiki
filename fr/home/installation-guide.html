Avant de commencer

  • BeamMP est compatible uniquement avec les versions officielles, Steam, de BeamNG.drive.
  • Les mods fonctionnent uniquement sur les serveurs sur lesquels ces derniers sont installés. Tenter d'utiliser un mod qui n'est pas autorisé par un serveur pourra résulter en un banissement.

Installation sur Windows

  1. Rendez vous sur notre site officiel et téléchargez le fichier BeamMP_Installer.zip le plus récent.
  2. Téléchargez soit Winrar soit 7zip. Un de ces deux programmes sera nécessaire pour la suite.
  3. Décompressez le fichier BeamMP_Installer.zip n'importe où. Pour ce faire, faites un clic droit sur le fichier .zip  puis Extraire ici.
  4. Lancez BeamMP_Installer.exe et suivez les instructions.
  5. Une icone BeamMP devrait apparaître sur votre bureau à la fin de l'installation, double cliquez dessus.
  6. C'est fini ! Appuyez simplement sur Jouer puis Multiplayer
  7. Il vous sera demandé de vous connecter, vous pouvez soit vous connecter en tant que visiteur ou bien créer un compte sur notre forum ici ! Attention, certains serveurs n'acceptent pas les comptes visiteurs.
  8. Cliquez sur le serveur qui vous intéresse le plus et cliquez sur Connect !

Problèmes connus:

  • Les caractères spéciaux dans votre nom d'utilisateur pourraient ne pas apparaître aux yeux des autres joueurs. S'il vous plaît assurez vous que votre nom d'utilisateur ne contient pas les caractères [ ]  ;  /  \  ou tout autre caractère spécial exotique.
  • Si vous ne voyez pas le bouton Multiplayer, essayez d'appuyer sur CTRL puis L, en même temps. Ensuite appuyez sur F5.

Pour toute autre aide vous êtes la bienvenue sur notre forum !

Bon jeu !

Installation sur Linux

  • BeamMP devrait fonctionner avec Wine, cependant le processus d'installation varie selon les distributions, nous vous invitons à vous renseigner sur l'installation de Wine pour votre distribution.
